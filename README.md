# Hotel Template

This repository provides a minimal setup for developing a hotel management application using React and Vite. It includes Hot Module Replacement (HMR) for a smooth development experience and ESLint rules for maintaining code quality.

## Features

- **Minimal Setup**: Quickly get started with React and Vite.
- **Hot Module Replacement (HMR)**: Enjoy a fast development experience with real-time updates.
- **ESLint Integration**: Enforce code quality and consistency with pre-configured ESLint rules.

## Technology Stack

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool that provides a modern development environment.
- **Babel**: A JavaScript compiler that allows you to use next-generation JavaScript today.
- **SWC**: A super-fast JavaScript/TypeScript compiler.

## Getting Started

To get started with this template, follow the steps below:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/nebaw-21/hotel_template.git
   cd hotel_template
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000` to see your application in action.
